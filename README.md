# 시도별 응급 의료 접근성 분석 및 클러스터링



## 1. 프로젝트 개요 및 분석 목표

### 프로젝트 명: AI/ML 기반 지역별 응급 의료 취약성 유형화 연구



본 프로젝트는 기존의 단순 통계 보고서의 한계를 극복하고, AI/ML (K-Means 군집 분석)을 활용하여 대한민국 17개 시도의 구조적 응급 의료 취약성 유형을 명확히 분류하는 것을 목표로 한다.





### 핵심 성과

데이터 전처리 및 다차원 클러스터링 분석을 통해, 획일적인 정책 적용이 불가능한 3가지 고유한 지역 유형을 도출하고 정책적 근거를 제시한다.

* 데이터 전처리 활동을 통해 인구/응급기관 데이터를 통합하고 5가지 핵심 지표를 구축하여 AI/ML 모델의 입력 데이터(Features)를 확보.

* AI/ML 분석 활동을 통해 K-Means 클러스터링 및 PCA를 수행하여 지역별 맞춤 정책 수립의 근거가 되는 유형을 도출.



---------------------

## 2. 핵심 분석 과정 및 방법론

### 가. 데이터 전처리 및 특성 공학 (Feature Engineering)

프로젝트의 첫 단계는 이종 데이터의 정확한 통합 및 분석 지표 구축한다.

* 데이터 통합 및 정규화: 인구 데이터에서 총인구 및 고령화율 지표를 산출했으며, 기관 데이터는 시도명 불일치 문제를 해결하고 컬럼을 정규화. 최종적으로 이 두 데이터를 통합하여 인구 10만 명당 4가지 기관 접근성 지표 (권역센터, 지역센터, 지역기관, 응급시설)을 계산.

* AI/ML 입력 데이터 확정: 클러스터링의 입력 변수로 고령화율과 4가지 인구 10만 명당 접근성 지표를 확정하여 총 5가지 다차원 벡터를 구성.



### 나. K-Means 군집 분석 (AI/ML Core Activity)

수행된 K-Means 클러스터링 분석은 다음의 논리로 진행되어 3가지 유형을 도출한다.

* 데이터 표준화: 5가지 지표 간의 척도 차이를 제거하기 위해 StandardScaler를 사용하여 데이터를 표준화.

* K-Means 실행: K=3으로 설정하여 17개 시도를 유형화.

* 결과 해석: 히트맵 분석을 통해 클러스터별 특징을 통계적으로 입증했으며, PCA 시각화를 통해 5차원 분석 결과가 2차원 평면상에서 명확히 구분됨을 확인.

## 3. 분석 결과 및 정책적 함의

### 가. 지역별 구조적 취약성 유형 도출

분석 결과, 17개 시도는 다음 3가지 고유한 유형으로 구분되어 획일적인 정책의 한계를 명확히 보여준다.

1. Cluster A: 고령/시설중심형: 고령화 압력이 최우선 해결 과제이며, 자원 효율화 정책이 필요.

2. Cluster C: 대도시/인구과밀형: 인구 과밀로 인한 자원 절대 부족이 문제이며, 상위 응급센터 확충이 시급.

3. Cluster B: 중위/균형형: 모든 지표에서 중간 수준을 유지하며, 현 시스템 유지 및 지역별 격차 해소에 집중 필요.



### 나. 향후 AI 기반 시스템 확장 논리

본 분석 결과는 향후 AI 시스템 구축의 핵심 근거가 됨

* AI 정책 시뮬레이터 설계 근거: K-Means분석으로 도출된 클러스터 중심점(Centroids)은 각 지역 유형의 최적 정책 목표를 수치적으로 정의하는 기준점이 된다. 시뮬레이터는 이 중심점을 기준으로 정책 효과를 예측하게 된다.

* AI 챗봇 설계 근거: 챗봇은 이 3가지 유형의 특징을 기반으로, 정책 결정자에게 구체적인 자원 배분 목표를, 일반 국민에게 실생활 응급 대비책을 맞춤형으로 제공하는 논리적 기반을 확보한다.
